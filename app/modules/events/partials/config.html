Add Extractor!

On left side you can select your python script to upload it and add a new data source to the Events Manager. The script needs to fulfill the following requirements:

  • A run()-method has to be implemented which needs to include the following parameters in the following order:

    • run(start, end, keyword)

    • through start and end you need to be able to pass a parameter in the format YYYY-MM-DD

    • through keyword you need to be able to pass a string as a parameter


  • The script has to return a python dictionary with the following keys:

    • Mandatory:

      • title as string

      • date in the format YYYY-MM-DDThh:mm:ssZ (e.g. 1954-07-17T00:00:00Z)

    • Optional:

      • description as string

      • url as string

      • enddate as string

      • keywords as string

      • geolocation as string

      • language as string


The python libraries 'SPARQLWrapper' and 'requests' are available for the Event Manager, so you are able to use and import them within your script.

After uploading the system will validate the script. If the validation is successful a new source will be available through the Events Manager!
