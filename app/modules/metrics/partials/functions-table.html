What is a normalisation function?

When comparing datasets the problem can occur that the values within those datasets are in different ranges. Imagine you have datasets from different sources containing the happyness-index for different coutries. Then the problem might be, that in one dataset the happyness-index ranges from 0 to 1 but in the other one it ranges from 0 to 100. A normalisation function can then be used to map values from one dataset to the range of the other dataset. For further information read the entry on wikipedia.
